# Setup

Install softwares for your Ubuntu 16.04 . 


### Installing

This script must be run as root 

```
sudo -s
```

## Software list

* Git 
* Gimp
* paper-gtk-theme
* paper-icon-theme
* unity-tweak-tool 
* openvpn 
* network-manager-openvpn 
* Discord 
* Chromium-browser
